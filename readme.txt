    Features:
-------------------------------------------------------

- Widgets
nakedpress has one sidebar on the right.

- Page Templates
nakedpress has a home page template with one slider that shows the latest three articles.

We Have:
Blog Layout, Category, Tag & Search Layout,

- Menus
One menu on the top.


-------------------------------------------------------
    Framework:
-------------------------------------------------------

nakedpress use bootstrap grid, menu and slider, also use font awesome for the icons.
